# 旅財布(tabi-budget)

旅行ごとに「予算を立てる → 使った金額を記録する → 予算と費用を比較する」ための個人用PWAアプリ。

## 特徴

- 📱 **PWA** — ホーム画面に追加してアプリのように使える。オフラインでも記録OK
- 💰 **予算づくり** — 日付ごと・単価×数量・一括、どんな立て方でも1つのフォームで
- 📝 **費用のクイック入力** — カテゴリ・金額だけの最小タップで記録。割引はマイナス入力
- 📊 **予算と費用の比較** — カテゴリ別の使用率をバーで表示。超過は色で警告
- 💱 **多通貨対応** — 通貨ごとに「予算レート」「費用レート」を分けて登録、日本円換算を自動計算
- 💵 **現金管理(海外旅行向け)** — 両替・キャッシングと現金払いを通帳スタイルで残高管理
- 💾 **JSONバックアップ** — 全データのエクスポート/インポート(機種変更・復元用)
- 📤 **スプレッドシート同期** — Google Apps Script経由で費用記録を一方向追記。家族は閲覧するだけ

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | アプリ本体(単体HTML) |
| `manifest.json` / `sw.js` / `icons/` | PWA関連 |
| `docs/SYNC_SETUP.md` | スプレッドシート同期の導入手順 |
| `docs/gas/Code.gs` | Google Apps Script側のコード |

## データの置き場所

すべてのデータは端末内(IndexedDB)にのみ保存されます。サーバーには何も置きません。
スプレッドシート同期を設定した場合のみ、費用記録があなたのGoogleスプレッドシートに追記されます。
同期のURL・合言葉は設定画面から入力し、端末内にのみ保存されます(コードに含まれません)。
