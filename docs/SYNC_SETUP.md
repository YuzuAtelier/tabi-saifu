# スプレッドシート同期の導入手順

旅財布の費用記録をGoogleスプレッドシートに自動追記し、家族が閲覧できるようにする手順です。
所要時間は10分ほど。Google Cloud ConsoleやOAuthの設定は不要です。

## しくみ

```
旅財布(スマホ) --POST--> GAS Webアプリ --追記--> スプレッドシート <--閲覧-- 家族
```

- 書き込むのはあなたの端末だけ。家族はスプレッドシートを「閲覧者」として見るだけです
- リクエストには合言葉(トークン)を付け、GAS側で照合します
- 合言葉とURLはアプリの設定画面から入力し、端末内にのみ保存されます

## 手順

### 1. スプレッドシートを作る

1. [Google スプレッドシート](https://sheets.new) で新規シートを作成
2. 名前を「旅財布の記録」など分かりやすいものに変更

### 2. Apps Script を設置する

1. シートのメニューから **拡張機能 → Apps Script** を開く
2. エディタに最初から入っているコードを全部消し、`docs/gas/Code.gs` の内容を貼り付ける
3. 💾(保存)を押す

### 3. 合言葉(トークン)を設定する

1. Apps Script エディタ左側の **⚙️ プロジェクトの設定** を開く
2. 一番下の **スクリプト プロパティ → スクリプト プロパティを追加** をクリック
3. 次のように入力して保存:
   - プロパティ: `SYNC_TOKEN`
   - 値: 好きな合言葉(例: `tabi-2026-himitsu`。長めのランダムな文字列がおすすめ)

※合言葉をコードに直接書かないための仕組みです。

### 4. Webアプリとして公開(デプロイ)する

1. エディタ右上の **デプロイ → 新しいデプロイ** をクリック
2. ⚙️(種類の選択)から **ウェブアプリ** を選ぶ
3. 次のように設定:
   - 説明: 任意(例: 旅財布同期)
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**
     (URLと合言葉の両方を知らないと書き込めないので、実質あなたの端末だけです)
4. **デプロイ** を押し、承認を求められたら自分のGoogleアカウントで許可する
   (「このアプリは確認されていません」と出たら「詳細」→「(プロジェクト名)に移動」で進めます)
5. 表示された **ウェブアプリのURL**(`https://script.google.com/macros/s/…/exec`)をコピー

### 5. アプリに設定する

1. 旅財布を開き、**⚙️設定 → スプレッドシート同期** へ
2. コピーしたURLと、手順3で決めた合言葉を入力して **保存する**
3. **接続テスト** を押して「接続OK!」と出れば完了 🎉
4. **今すぐ同期する** を押すと、未同期の費用記録がシートに書き込まれます

### 6. 家族に共有する

1. スプレッドシート右上の **共有** から家族のGoogleアカウントを追加
2. 権限は **閲覧者** にする(家族は見るだけ。編集はできません)

## 日々の使い方

- 費用を記録すると、オンラインなら自動でシートに追記されます(操作不要)
- 圏外・機内モードで記録した分は「未同期」として貯まり、オンライン復帰時に自動送信されます
- 未同期の件数は 設定 → スプレッドシート同期 で確認できます
- 記録をあとから編集した場合は、シートの同じIDの行が上書きされます(行が二重になることはありません)

## うまくいかないとき

| 症状 | 確認すること |
|---|---|
| 接続テストが失敗する | URLが `/exec` で終わっているか。「アクセスできるユーザー: 全員」になっているか |
| 「拒否されました」と出る | アプリの合言葉とスクリプトプロパティ `SYNC_TOKEN` が完全に一致しているか |
| コードを直したのに反映されない | **デプロイ → デプロイを管理 → ✏️ → バージョン: 新バージョン** で更新する(新しいデプロイを作るとURLが変わるので注意) |
| シートに変な行が増えた | シート側は自由に編集・削除してOK(アプリには影響しません) |
